## 🚀 Getting Started

### Prerequisites
```bash
Flutter SDK: ^3.8.1
Dart SDK: ^3.8.1
```

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/escape.git

# Navigate to project
cd escape

# Install dependencies
flutter pub get

# Run the app
flutter run
```
